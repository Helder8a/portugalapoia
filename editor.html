Editor Visual de Contenido

Edita el contenido de tu web aquí. Cuando termines, haz clic en el botón grande al final para generar el código, cópialo y pégalo en tu archivo contenido.js.

Generar Código para `contenido.js`

1. Haz clic en el botón de abajo.
2. El cuadro de texto se llenará con el código actualizado.
3. Haz clic derecho en el cuadro -> Seleccionar todo -> Copiar.
4. Pega el código en tu archivo contenido.js, reemplazando todo el contenido antiguo.
